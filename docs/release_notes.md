# Release Notes

## Version 1.0.0

Initial release of StackVPN - Open-source dual-protocol VPN client for Android.

### Features
- Dual VPN support (WireGuard + OpenVPN)
- No root required
- Tracker and ad blocking capabilities
- Kill switch functionality
- Split tunneling support
- Custom DNS resolver

### Security Features
- No logging policy
- DNS leak protection
- Kill switch implementation
- Split tunneling support

### Technical Specifications
- Supports Android 8.0+ (API 26+)
- Built with Kotlin and Jetpack Compose
- Open source under GPL-3.0 license
