# User Guide

## StackVPN User Guide

This guide provides instructions on how to use the StackVPN application.

### Getting Started

1. Install the StackVPN application on your Android device (8.0 or higher)
2. Launch the application
3. Grant the necessary permissions when prompted
4. Configure your VPN profiles
5. Connect to your desired VPN service

### Main Features

#### Dual VPN Support
StackVPN allows you to chain multiple VPN protocols in a single application, providing enhanced privacy and security without requiring root access.

#### Profile Management
- Create and manage multiple VPN profiles
- Configure WireGuard and OpenVPN settings
- Save connection preferences

#### Security Features
- Kill switch to block traffic when disconnected
- DNS filtering to block ads and trackers
- Split tunneling to exclude specific apps from VPN routing

### Usage Instructions

#### Connecting to VPN
1. Open the StackVPN app
2. Select your desired profile
3. Tap the "Connect" button
4. Grant VPN permissions when prompted
5. The app will establish a secure connection

#### Configuring Profiles
1. Navigate to the Profiles screen
2. Tap the "+" button to create a new profile
3. Enter the configuration details for your VPN service
4. Save the profile

#### Settings
1. Access the Settings screen
2. Configure security features like kill switch and DNS filtering
3. Set up split tunneling for specific applications
4. Configure automatic connection settings

### Troubleshooting

#### Connection Issues
- Check your internet connection
- Verify your VPN configuration
- Try reconnecting
- Check if kill switch is blocking traffic

#### Performance Issues
- Restart the application
- Check battery optimization settings
- Ensure sufficient device memory

### Privacy and Security

StackVPN is designed with privacy and security in mind:
- No logging of user activity
- No data collection without consent
- Open source code for transparency
- Strong encryption protocols
- Kill switch functionality
- DNS leak protection
